Cui-Avatar
Cui-avatar is a directive that inserts the user's avatar into the background of the div it's applied to. This can be used in conjunction with off-click to create a login / logout box, like this one. Click here for the code for this page.
Hello, {{app.appUser.name}} !!
Logged in as {{app.appUser.id}}.
