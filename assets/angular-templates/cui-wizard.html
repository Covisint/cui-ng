Cui-Wizard
Cui-wizard is an angular directive that, following a few syntax rules, allows the developer to easily create a multi-step form or any other component that requires a "step-through" approach.

Click here for the extended documentation.
Anything can be put inside of these steps - from static html content to rich dynamic forms. If you are using this for forms you can use the nextWithErrorChecking() function to help you out!
