Cui-Icon

The cui-icon directive makes it easier to use vector-based icons in your app (as opposed to raster-based icons types like PNG).

The directive supports both icon fonts and SVG icons.
All styling (color, size, etc) can be done as expected, via css.




This directive requires a bit of setup to be able to use, refer to the docs for more info.
