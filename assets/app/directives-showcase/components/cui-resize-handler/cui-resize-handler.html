Cui-Resize-Handler

Cui-Resize-Handler allows you to only show certain elements in either a desktop or mobile view.

The caveat is that elements that are not supposed to be show are completely removed from the DOM. This prevents situations where you have a desktop and mobile view tied to the same controller which would end up firing the controller twice (once for each view).

The "breakpoint" (or screensize where something is considered mobile/desktop) can be set specifically on the element, or it can be configured in your project appConfig.

Try it out: resize the screen to show a mobile or desktop Murray. The first Murrays are set through the appConfig, the second on the element themselves.
