This dropdown uses the default type-ahead keyboard functionality. You can select dropdown items by typing one of the options on your keyboard. You can press the Delete button to clear the type-ahead search.

This dropdown allows you to search/toggle through the options based on the first letter of each option. To use this simply add the single-search attribute to the Cui-Dropdown element.

The dropdowns also support all other default keyboard functionality. If you select the dropdowns with tab-targeting, you can open them using the Up/Down/Space/Enter keys. You can cycle through the options using the Up/Down arrow keys. You can select options using the Space/Enter keys. You can also close a dropdown by tabbing out of it.
