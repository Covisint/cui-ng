Auto-Complete
Auto-complete is a directive that enables type-ahead for any type of data - server or client side.

Below is a country input box, try it out!
You've selected {{autoComplete.selectedCountry.originalObject.name}}. The code for this country is {{autoComplete.selectedCountry.originalObject.code}}.

Changing the language in the dropdown will allow you to search for countries in that language. This data can be updated in real time if triggered by a broadcast or a $watch.
